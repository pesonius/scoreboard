User Guide

Quick Start

  • Add players to the roster, then tap to select them.
  • Choose Singles or Doubles format.
  • Choose points to win (21 or 11).
  • Set your clicker buttons under Key Mapping.
  • Tap Start Match — the scoreboard opens full-screen.

Clicker Controls

Short press Score a point for that player/team.
Long press (≥ 0.8 s) Undo the last point. Hold until the score flashes red.
Button 1 Point for the left side.
Button 2 Point for the right side.

The golden dot (●) indicates the current server. Up to 50 undo steps are stored — they survive a page refresh.

iPad Setup

  • Connect your Bluetooth clicker to the iPad — it must produce keyboard events (arrow keys, letters, or space; volume keys are intercepted by iOS).
  • For full-screen: tap Share → Add to Home Screen, then open from the home screen icon.

Session Mode

  • Games are unlimited — play as many as you want.
  • After each game an overlay shows the winner and running tally. Press any button to start the next game.
  • Each game is saved to history individually.
  • Press End on the scoreboard when done.

Scoring Rules

  • First to 21 (or 11), win by 2, max 30 (or 20).
  • Rally point — winner of each rally serves next.

Season Stats

  • Export downloads a JSON backup of all history.
  • Import merges a backup — duplicates are skipped.
