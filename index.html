User Guide

Quick Start

  • Add players to the roster, then tap to select P1 and P2.
  • Choose a sport and match format.
  • Set your clicker buttons under Key Mapping — press Set Key then press the physical button.
  • Tap Start Match — the scoreboard opens full-screen.

Clicker Controls

Short press Score a point for that player. Press immediately and release.
Long press (≥ 0.8 s) Undo the last point. Hold until the score flashes red, then release.
Button 1 Point for the left player (P1).
Button 2 Point for the right player (P2).

The golden dot (●) next to a name indicates the current server. Up to 50 undo steps are stored — they survive a page refresh.

iPad Setup

  • Connect your Bluetooth clicker to the iPad — it must produce keyboard events (arrow keys, letters, or space work; volume keys are intercepted by iOS).
  • Open this page in Safari on the same WiFi network using the Mac's local IP address and port 3000.
  • For true full-screen (no Safari chrome): tap the Share button → Add to Home Screen, then open from the home screen icon.

Scoring Rules

Badminton First to 21 (or 11), win by 2, max 30 (or 20). Rally point — winner serves. Service changes at 11 in the deciding game.
Tennis 0 / 15 / 30 / 40 / Deuce / Advantage. First to 6 games (win by 2); tiebreak at 6-6. Server alternates each game.
Squash First to 11, win by 2, no maximum. Rally point — winner serves. Best of 5.
Table Tennis First to 11, win by 2, no maximum. Service alternates every 2 points (every 1 point from 10-10).

Badminton Session Mode

  • Select Session under Mode to play unlimited games — no match limit.
  • After each game, an overlay shows the winner and running tally. Press any clicker button to start the next game immediately.
  • Each game is saved to history individually as it finishes.
  • Press End Session in the scoreboard header when done — takes you straight to Season Stats.

Match End (Best Of mode)

  • Rematch — swaps who serves first, resets scores, keeps the same players.
  • New Match — returns to this setup page.
  • Completed matches are saved automatically and appear in Season Stats.

Season Stats

  • Filter standings and head-to-head by sport using the dropdown.
  • Export downloads a JSON backup of all players and match history.
  • Import merges a backup file — duplicate matches are skipped, so you can import from multiple devices safely.
